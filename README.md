# **R Programming Workshop Materials - GDSS26**

This repository contains materials used during my R programming workshop.

## **Structure of the materials**

### 1. PowerPoint Slides

Provides an overview of R programming, its importance, R and its application in Research and AI

### 2. R Scripts

There are two R scripts:

- Ghana R - 50 Days.R - This contains introductory concepts in R programming, covering basic commands and foundational knowledge. Participants are advised to go through this file first to be able to understand the scripts in the workshop file.

- WorkshopFile.R - This script builds on the introductory content and focuses on the main workshop exercises and applications.

### 3. Dataset

student_performance_dataset.csv - This dataset was generated purely for practice purposes and is not real. Participants can use it to:

- Practice what they have learnt

- Explore basic statistical operations

- Understand how the statease shiny app interacts with datasets.
